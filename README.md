# Made a good Video Player Using JavaScript DOM interface and complete able to work on live server

You can modify the static path of the media and run it it play nice 


**Features:**
1. thumbnail uploads
2. automatically thumbnail fetches from video element using canvas element to draw exact frame image
3. ambilight effect on play
4. Screenshot of videos
5. skip
6. 2X speed toggle
7. Speed selector
8. explicit subtitle uploads
9. double click to maximize feature
10. skip buttons
11. right click on them make it customize their ability( skip buttons and 2x toggler)
12. clip extraction
13. Picure In Picture toggler
14. 3s GIF creation
